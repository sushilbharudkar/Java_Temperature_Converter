# Java Temperature Converter

## Description
This is a Java console-based mini project that converts temperatures between
Celsius, Fahrenheit, and Kelvin using a menu-driven approach.

## Key Features
- Converts Celsius, Fahrenheit, and Kelvin
- Menu-driven user interface
- Simple and beginner-friendly
- User input using Scanner class
- Uses loops and switch-case statements

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- Scanner class

## How to Run
1. Clone or download the repository
2. Compile the program:
   javac TempConverter.java
4. Run the program:
   java TempConverter
